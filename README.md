# MySQL Web Editor #

## Description ##
MySQL Web SQL editor

## Requirements ##
* [PHP 5.4.1 or higher](http://www.php.net/)
* [CodeMirror](https://codemirror.net/)

## Installation ##
~~~
1.- Clone repository
2.- Create config.php
~~~

## Example ##
~~~
<?php
$dbHost = "localhost";
$dbUser = "mysql_user";
$dbPwd = "mysql_password";
$dbName = "mysql_database";
$dbPort = 3306;

$appUsr = "app_user";
$appPwd = "app_secure_password";
?>
~~~
